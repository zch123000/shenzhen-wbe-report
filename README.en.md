# SARS-CoV-2 Wastewater-Based Epidemiology Report — Shenzhen WBE Project

## SARS-CoV-2 Nucleic Acid Testing Analysis of Shenzhen Municipal Wastewater Treatment Plants

A study on SARS-CoV-2 detection positivity rates in Shenzhen municipal wastewater treatment plants using the iMB-DE method, with comparative analysis against KT/MT methods.

### Contents

- **Positivity Rate Trend Analysis** — Time-series comparison of positivity rates across sampling sites
- **Methodology Comparison** — Detection performance comparison of iMB-DE vs KT vs MT methods
- **Geographic Heatmap** — Spatial distribution of positivity rates across Shenzhen districts
- **Limit of Detection Analysis** — Method sensitivity validation

### Tech Stack

- Chart.js 4.4 — Data visualization
- Leaflet 1.9.4 — Geographic heatmap
- Pure HTML/CSS/JS — No build tools required
